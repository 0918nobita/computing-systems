# BASIC Language Compiler

## Target

x86_64 Linux

## Requirements

- Cargo
- GNU Binutils
- NASM

## Usage

```bash
cargo run -- ../samples/basic/hello.bas  # outputs ../basic/hello.bin
../samples/basic/hello.bin  # => Hello, world!
```

## Build

```bash
cargo build
```

## Generate documentation

```bash
cargo doc
```

## Clean

```bash
cargo clean
```
